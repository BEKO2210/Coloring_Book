# Step 6 — KDP Listing (Amazon.de)

**Buch:** Große Mandalas für ruhige Stunden
**Markt:** amazon.de • **Kategorie-Ziel:** Coloring Books für Senior:innen

---

## 1️⃣ SEO Title (für Amazon-Produkttitel)

**Empfohlen (Länge 34 Zeichen):**
```
Große Mandalas für ruhige Stunden
```

→ Hauptkeyword `Mandala` front-loaded nach Emotion-Anker `Große`. Der Rest der Keyword-Arbeit findet im Subtitle statt (Amazon erlaubt Kombi Title+Subtitle bis 200 Zeichen).

---

**Empfohlen (106 Zeichen):**
```
Das Mandala-Malbuch für Senioren mit extra großen Motiven — auch für Menschen mit Demenz geeignet
```

| Keyword | SEO-Relevanz |
|---|---|
| Mandala Malbuch | ⭐⭐⭐⭐⭐ Hauptsuche |

**Gesamt Title+Subtitle: 140 Zeichen / 200 erlaubt** ✅

### Regeln befolgt ✅
- Jede Zeile ≤ 50 Zeichen
- Keine Duplikate über alle 7 Zeilen
- Keine Markennamen / keine Konkurrenten
- Keine medizinischen Claims (kein „heilt", „gegen Alzheimer")
- Sinnvolle Keyword-Kombis, nicht nur Komma-Listen

## 7️⃣ Kategorien (Amazon.de Browse Paths)

KDP erlaubt 2 Hauptkategorien (bzw. 3 über Kontaktformular).

### Primäre Kategorie (stärkste Relevanz)
```
Bücher > Ratgeber > Gesundheit & Medizin > Alter & Pflege
```
→ Riesige Demografie-Überschneidung, relativ wenig Mandala-Konkurrenz.

### Sekundäre Kategorie
```
Bücher > Hobby, Haus & Garten > Zeichnen & Malen > Ausmalbücher für Erwachsene
```
→ Hauptmarkt für Coloring Books auf amazon.de, Haupt-BSR-Ranking.

### Optionale dritte Kategorie (via KDP-Support anfragen)
```
Bücher > Ratgeber > Lebensführung & persönliche Entwicklung > Meditation & Selbstfindung
```
→ Trifft die „Achtsamkeits"-Käuferschaft.

## 🎯 Launch-Playbook (Tag 0–30)

### Tag 0 — Upload
- [ ] KDP-Konto → Paperback anlegen
- [ ] Interior-PDF hochladen (68 S., 8,5×11", No-Bleed, s/w)
- [ ] Cover-PDF hochladen (mit KDP Cover-Template generieren)
- [ ] Preis 9,99 € (Launch-Tarif)
- [ ] Alle oben stehenden Listing-Daten einpflegen
- [ ] Buch einreichen (Review-Zeit 24–72 h)

### Tag 1–3 — Go Live
- [ ] Status-Check: Buch „live" auf amazon.de
- [ ] Selbst bestellen (1 Exemplar) → Druckqualität prüfen
- [ ] Freunde/Familie um 3–5 **ehrliche** Erstbewertungen bitten (keine gekauften!)

### Tag 4–14 — Launch-Phase
- [ ] Preis auf Launch-Tarif 9,99 € halten
- [ ] Amazon-PPC starten mit **5 €/Tag Budget**, nur auf Backend-Keywords #1, #3, #6
- [ ] Social Teaser posten (Foto vom Buch in ruhiger Umgebung, Hook-Variante 1)

### Tag 15–30 — Konsolidierung
- [ ] Preis auf regulären Preis **11,99 €** anheben (wenn ≥ 10 Reviews)
- [ ] BSR-Tracking: Ziel < 80.000 in Kategorie „Alter & Pflege"
- [ ] PPC-Budget anpassen: Keywords mit ACoS < 35% skalieren
- [ ] **Nächstes Buch starten** (Serie: Buch 9 „XL-Mandalas" nach Launch-Plan)
